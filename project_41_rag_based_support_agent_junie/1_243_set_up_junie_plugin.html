What is Junie

Junie is part of the JetBrains AI suite. While the basic AI Assistant can already generate text or code snippets, Junie takes it up a notch by using project context. That means she's not just smart—she's aware of what you're working on.

With Junie, you can:

  • Delegate. Iterate. Review – Assign tasks to Junie, and it will handle them autonomously. Junie will save you time and effort that you can use to drive results and creative solutions instead.
  • Stay in control of your code – Share prompts in the chat with Junie, review the results, and accept or iterate as needed. Adjust, experiment, and create with better focus and fewer distractions.
  • Keep the project consistent – Junie learns the context of your code, understanding your preferences and style. Refining its approach with each iteration, Junie helps you write cleaner, more consistent code.
Junie currently supports JetBrains IDEs like IntelliJ IDEA, PyCharm, WebStorm, and others.
Installing

Installing Junie is straightforward, and you only need a few clicks:

  1. Go to Settings / Preferences → Plugins.
  2. Search for Junie and click Install.
  3. Restart the IDE when prompted.

Once the plugin is active, Junie becomes available through the IDE.

Make sure you're logged in with your JetBrains account that has access to the AI features.
