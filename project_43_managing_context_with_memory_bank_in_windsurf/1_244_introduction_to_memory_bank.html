What is Memory Bank?

Memory Bank is a structured documentation system that helps Cascade maintain context across sessions. It transforms Cascade from a stateless assistant into a persistent development partner that can effectively "remember" your project details over time.

When working with AI coding, you've likely encountered this common limitation: the AI has no memory of your previous conversations once you start a new session. This means:

  • You need to re-explain your project repeatedly
  • The AI can't recall decisions you made together
  • You lose continuity in complex projects
  • Each session starts from scratch
How Memory Bank Solves This Problem

Memory Bank addresses this limitation through a systematic approach to documentation that complements Cascade's native Memories system:

  • Structured Documentation: A hierarchy of markdown files that build a complete picture of your project.
  • Context Preservation: Key information is stored in a format that's easy for both you and Cascade to reference.
  • Continuous Updates: Documentation evolves as your project progresses.
Memory Bank is Not Magic

It's important to understand that Memory Bank isn't a magical feature that gives AI Agent perfect memory. It's a methodical approach to documentation that you maintain to enhance the AI's understanding. Cascade already has powerful features like code indexing and its "Memories" system to retain context. Our Memory Bank structure will supercharge these capabilities.

Setting Up Memory Bank

To set up Memory Bank we will use Memory Bank MCP Server. It's a simple way to create a Memory Bank for your projects automatically based on provided context.

First of all, we need to install uv and uvx tools to run Memory Bank MCP Server.

Install uv and uvx tool if it's not installed yet

Then, we need to configure Memory Bank MCP Server.

Open Windsurf settings:

  • For Mac: Windsurf → Settings → Windsurf Settings or CMD⌘ + ,
  • For Windows: Settings → Windsurf Settings or CTRL + ,

Then open Cascade tab and press the manage plugins button. You should see the following screen:

Now press View raw config button and add Memory Bank MCP to the list of MCP servers:

{ "mcpServers": { "mcp-memory-bank": { "command": "uvx", "args": [ "--from", "git+https://github.com/ipospelov/mcp-memory-bank", "mcp_memory_bank" ] } } }

Alternatively, you can use Docker to run Memory Bank MCP Server.

Setup with Docker { "mcpServers": { "memory-bank": { "command": "docker", "args": [ "run", "-i", "--rm", "19283744/mcp-memory-bank:latest" ] } } }

After that, you should see the Memory Bank MCP Server in the list of your plugins with the following tools:
