The Playwright MCP is a powerful server that brings Microsoft's Playwright browser automation capabilities to Windsurf. This extensive toolkit enables precise control over web browsers, allowing for sophisticated web interactions, testing, and automation.

Features:
  • Comprehensive browser control with multiple interaction modes
  • Keyboard and mouse control for complex interactions
  • Screenshot capabilities with customization options
  • Tab and window management for multi-page workflows

Key Use Cases
  • Automated web testing and verification
  • Complex form filling and submission
  • Web scraping with precise element targeting
  • Multi-step workflow automation
  • Cross-browser compatibility testing

You can set up the Playwright MCP using NPX, which is the recommended method:

NPX Setup
{ "mcpServers": { "playwright": { "command": "npx", "args": [ "@playwright/mcp@latest" ] } } }

For headless browser operation (no visible browser window), use:

{ "mcpServers": { "playwright": { "command": "npx", "args": [ "@playwright/mcp@latest", "--headless" ] } } }
