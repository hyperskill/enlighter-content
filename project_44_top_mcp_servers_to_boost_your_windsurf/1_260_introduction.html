What are MCP Servers?

MCP (Model Context Protocol) servers are modular tools that connect to your AI Agent and provide it with additional context and tools to achieve better results in writing code, debugging, and more.

What You'll Learn
  • How to install MCP servers in Windsurf
  • Various types of MCP installations
  • Top MCP servers to boost your productivity and results in Windsurf
  • How to find new MCP servers and tools for your needs
Windsurf Plugin Store

Windsurf plugin store is a collection of MCP servers and tools that you can easily add to your Windsurf with one click. If MCP server is available in the plugin store, we will mention it in the description and recommend you to install it from the plugin store.

However, not all of MCP servers in our list are available in the plugin store. That's why we will also show you how to install MCP servers manually.

NPX Installation

Throughout this project, we'll primarily use NPX to install and run MCP servers. NPX is included with Node.js and npm, which you'll need to install if you don't have them already:

Install npx, node, and npm. Verify your installation by running: ``` node --version npm --version npx --version ``` All three commands should display version numbers if the installation was successful.

Install uv and uvx tool if it's not installed yet
Docker (Optional)

Some MCP servers provide Docker installation options, which can be useful for containerized environments or if you prefer isolation. Throughout this project, we'll show Docker configuration examples alongside NPX options.

If you choose to use Docker, you'll need to install Docker Desktop or Docker Engine on your system. Installation instructions vary by operating system and are beyond the scope of this project. Please refer to the official Docker documentation for installation guidance.

Manual MCP Server Installation

Let's see how to install any MCP Server manually. We will use Memory Bank MCP Server as an example.

Open Windsurf settings:

  • For Mac: Windsurf → Settings → Windsurf Settings or CMD⌘ + ,
  • For Windows: Settings → Windsurf Settings or CTRL + ,

Then open Cascade tab and press the manage plugins button. You should see the following screen:

Now press View raw config button and add Memory Bank MCP to the list of MCP servers:

{ "mcpServers": { "mcp-memory-bank": { "command": "uvx", "args": [ "--from", "git+https://github.com/ipospelov/mcp-memory-bank", "mcp_memory_bank" ] } } }

After that, you should see the Memory Bank MCP Server in the list of your plugins with the following tools:

Well done! You have installed Memory Bank MCP Server manually.

Limitations
Having too many MCP servers and tools added to Windsurf can confuse Cascade Agent and make it less reliable in choosing correct tools. That's why Windsurf currently has a limit of 100 tools. Make sure to only enable the MCP servers and tools that you really need for your current project.
