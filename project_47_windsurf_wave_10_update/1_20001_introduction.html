Introduction

AI-powered development environments like Windsurf are evolving rapidly, introducing new features designed to enhance developer productivity. The primary goal of these advancements is to improve the contextual understanding of Large Language Models (LLMs), enabling them to generate more accurate and reliable code. This tutorial will explore some of these new capabilities.

The Windsurf Wave 10 Update

This tutorial focuses on the Wave 10 update for Windsurf, which introduces two significant features aimed at improving the developer's interaction with the AI assistant, Cascade. These features are:

  • Planning Mode: A tool designed to help manage long-term project goals and complex, multi-step tasks by maintaining a shared project plan.
  • The Windsurf Browser: An integrated, Chromium-based browser that provides the AI with direct visual and technical context of a web application's front end.
What You'll Learn

By following this tutorial, you will gain practical knowledge on how to:

  • Set up and use Planning Mode to create a structured project roadmap with an AI assistant.
  • Utilize the integrated Windsurf Browser for live application previews and context-aware front-end development.
  • Apply an effective workflow that combines both features to build a web application from an initial concept to a functional product.
  • Communicate with the AI more efficiently for tasks like styling, feature implementation, and debugging.
About the Learning Project

To demonstrate these features in a practical setting, we will build a "Smart Task Manager" web application. This project was chosen because its development cycle naturally showcases the benefits of the new tools. We will use standard web technologies (HTML, CSS, and vanilla JavaScript) to keep the focus on the IDE's workflow rather than on a complex framework.

You will learn how to:

  1. Use Planning Mode to outline the application's core features and structure at the beginning of the project.
  2. Refer to this plan throughout development to guide the AI in generating code step-by-step.
  3. Use the Windsurf Browser to render the application, provide visual context to the AI for styling, and debug front-end issues directly within the IDE.

This hands-on approach will help you understand how to integrate these features into your own development practices.

Getting Ready

To complete this tutorial, you will need Windsurf IDE version 1.10.0 (Wave 10) or later installed on your system. No other project files or specific configurations are required, as we will be building everything from scratch.

To check your version of Windsurf IDE, navigate to Windsurf > About Windsurf from the menu bar. If your version is older, please update it before proceeding.

Outcome of This Lesson and Next Steps

The objective of this lesson was to provide a clear understanding of the new features in Wave 10 and the goals of this tutorial. With the setup complete, we are ready to begin our project.
