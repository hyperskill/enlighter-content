The Memory Bank MCP provides a structured way to maintain project context for AI assistants. By organizing project information into categorized markdown files, Memory Bank helps Cursor better understand your project's goals, architecture, and current focus. This stage will guide you through setting up and effectively utilizing the Memory Bank MCP in your Cursor workflow.

Features:
  • Organize project context in hierarchical markdown files
  • Maintain persistent knowledge between Cursor sessions
  • Retrieve specific context information on demand
  • Generate template files based on project needs
  • Analyze project summaries for better context organization

Key Use Cases
  • Long-term project development with consistent AI assistance
  • Complex projects requiring detailed context understanding
  • Team collaboration where contextual knowledge needs sharing
  • Projects with evolving requirements and architecture

Memory Bank organizes project context into six core markdown files:

  1. projectbrief.md
    • Foundation document with core requirements
    • Defines project scope, goals, and success criteria
    • Created first and rarely changes

  2. productContext.md
    • The "why" of your project
    • Explains user needs and problems being solved
    • Outlines user experience goals

  3. systemPatterns.md
    • The "how" of your project
    • Documents system architecture and design patterns
    • Records key technical decisions

  4. techContext.md
    • Tools and technologies
    • Lists frameworks, libraries, and development tools
    • Documents technical constraints

  5. activeContext.md
    • Current project state
    • Describes what you're working on now
    • Records recent changes and next steps

  6. progress.md
    • Project status tracking
    • Lists completed features and known issues
    • Records the evolution of project decisions

This MCP helps to automatically create memory bank files in your project.

Setup Instructions

You can set up the Memory Bank MCP using SSE connection.

SSE Setup
Create or edit a file .cursor/mcp.json to include the following MCP setup: { "mcpServers": { "memory-bank": { "url": "https://mcp-cursor-setup-enlighter.replit.app/sse" } } }

You can also set up the Memory Bank MCP manually with GitHub repository.

Running the Server

You can run the server using the provided script:


#!/bin/bash

# Install dependencies if needed
if [ ! -d "venv" ]; then
    echo "Creating virtual environment..."
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
else
    source venv/bin/activate
fi

# Run the server
echo "Starting Memory Bank Helper MCP Server..."
python server.py

Or manually:


python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python server.py

The server will be available at http://localhost:8080/sse.

MCP .json setup example
{ "mcpServers": { "memory-bank": { "url": "http://localhost:8080/sse" } } }
