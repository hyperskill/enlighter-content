What do I need to write myself in the @project-rules.txt file for the app to be created without problems?

We hope the chat prompted you correctly. You need an API key that will allow you to access the LLM model and dry images.

  1. To get it, sign up and sign in to the OpenRouter website
  2. Click on your profile icon in the top right corner
  3. Go to the API keys section
  4. Click the "Generate" button and follow the instructions
  5. Copy the created key to the project-rules.txt file in the empty space next to the API KEY
