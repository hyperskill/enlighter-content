MCP Servers: One-Click External Tool Integration

Cursor 1.0 makes external tool integration with MCP (Model Context Protocol) servers even easier, offering one-click setup and OAuth support for seamless connections to external services and APIs.

What are MCP Servers?

MCP servers extend Cursor's capabilities by connecting to external tools and services:

  • Protocol Standard - Standardized way for AI to interact with external tools
  • One-Click Setup - Install and configure servers with minimal effort
  • OAuth Support - Secure authentication with supported services
  • Curated Selection - Official MCP servers vetted for quality and security

To add selected MCP servers, go to docs.cursor.com/tools and browse the catalog of available servers.

Other features of Cursor 1.0

The chat interface now supports sophisticated visualizations and formatted content:

  • Mermaid Diagrams - Generate and view flowcharts, sequence diagrams, and more
  • Markdown Tables - Rich formatted tables with sorting and styling
  • Code Syntax Highlighting - Enhanced code blocks with language-specific formatting
  • Interactive Elements - Clickable links, collapsible sections, and more

For example, you can ask the Agent to create a Mermaid diagrame for your codebase:

Generate a Mermaid diagram that visualizes my project structure"
