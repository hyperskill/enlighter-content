Now that we've built our Echo MCP server, let's configure Claude Desktop to use it and test its functionality. In this stage, we'll add our server to the MCP configuration, verify it's enabled in settings, and test each of its features.

Adding the Echo Server to MCP Configuration

First, we need to add our custom MCP server to Claude's claude_desktop_config.json file. The same as before, we need to add the server to the claude_desktop_config.json file.

{
  "mcpServers": {
    "Echo": {
      "command": "path/to/your/project/venv/bin/python3.12",
      "args": [
        "path/to/your/project/echo_server.py",
      ]
    }
  }
}
Don't forget to update the path to the python executable and the path to the echo_server.py file. Verifying MCP Server is enabled

Next, restart Claude Desktop and click on the hammer icon to check if the server is enabled as explained in the previous steps.
