Now that we've set up our Memory Bank, let's start using it to develop our TodoMaster app. We'll begin by creating the basic HTML structure.

Starting a Development Session

When starting a new development session with Cursor, first ask it to read your Memory Bank:

Please follow your custom instructions and read my Memory Bank files for the TodoMaster project. I'd like to start implementing the basic HTML structure.

Cursor will respond with something like:

"I've read your Memory Bank files for TodoMaster. I see this is a task management application using vanilla JavaScript with localStorage for data persistence. According to your activeContext.md, you're currently focusing on setting up the basic HTML structure. Let's get started on that."

Creating the HTML Structure

Now, ask Cursor to help implement the HTML structure based on your Memory Bank context:

Based on our Memory Bank, please help me create the basic HTML structure for our TodoMaster app. We need sections for: 1. Header with app title and add task button 2. Task list view 3. Task creation/editing form 4. Filter controls Updating Memory Bank

After implementing the HTML structure, update your Memory Bank:

I've implemented the basic HTML structure for TodoMaster. Please update our Memory Bank to reflect this progress.

Cursor will update the relevant files, particularly activeContext.md and progress.md.

Completion Checklist
  • Use the provided callout to start a session with Cursor and load your Memory Bank
  • Ask Cursor to help create the HTML structure for TodoMaster
  • Update your Memory Bank to reflect the progress made
  • Check your Memory Bank files to see how they've been updated
