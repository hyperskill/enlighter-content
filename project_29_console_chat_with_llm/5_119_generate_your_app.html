AI is still not perfect. And a successful result is already very likely, but not mandatory. After several attempts you always get what you need.

We have also noticed that sometimes the application is created but not launched. In this case, you just have to ask Chat: run my application and fix the errors that occurred at startup.

Also, there is a known issue when the Cursor app has expired its attempts to generate your app and it asks to continue manually. You will see this message: "Note: By default, we stop the agent after 25 tool calls. Please ask the agent to resume manually".

In this case, please open a new Chat and continue fixing the app. Don't forget to check if project-rules.txt is added to make the Chat context work properly.
