Before we start working on the app, please install JetBrains Junie plugin.

Installing Junie is straightforward, and you only need a few clicks:

  1. Go to Settings / Preferences → Plugins.
  2. Search for Junie and click Install.
  3. Restart the IDE when prompted.

Once the plugin is active, Junie becomes available through the IDE.

Make sure you're logged in with your JetBrains account that has access to the AI Assistant features.

So first, we recommend that you turn on YOLO mode in the cursor settings. You can find this option in Preferences, then Cursor Settings, then Features.

Set your Chat to Agent mode

These requirements will help the LLM to perform autogeneration and auto-fixing of your application with minimal user intervention. Sometimes you will be asked to run shell commands, but in most cases the cursor will do everything by itself. Chat will also see any errors that occur and try to fix them on the fly.

If you are unable to activate agent mode in an empty project, that's okay. Just make sure that agent mode is enabled before generating your application. Cursor tries to detect the type of your request automatically and does not always activate the right switch.
