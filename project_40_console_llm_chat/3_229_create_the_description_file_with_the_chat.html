Learning how to write clear descriptions is the most important part of working with code generation tools. Creating applications with AI is still software engineering — AI just helps along the way. The good news is that AI excels at explaining complex terms and concepts.

Put its abilities to the test: in the chat window, you can attach any file by pressing the "+" button and ask it to clarify something or perform a task based on the file's contents.

Adding files to context

Usually, Junie is quite good at navigating the project on his own but if you want Junie to be laser-focused, you can hit the "+" button to attach specific files or folders. This is how you give Junie access to custom parts of your project—like your README, API guidelines, or folder structures.

You can attach:

  • Config or certain files;
  • Entire directories (e.g., /src/controllers);
  • Single files with tricky logic or expanded instructions.

This is a huge help in larger projects. Instead of explaining everything to Junie, just show it.

Don't forget to add task-description.txt file to the context.
