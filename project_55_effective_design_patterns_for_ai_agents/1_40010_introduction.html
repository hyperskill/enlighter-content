Introduction

Welcome to the course on Effective Design Patterns for AI Agents. In this hands-on project, you will implement four foundational patterns that make agent systems more reliable, scalable, and safe. We will use Google's open-source ADK (Agent Development Kit) for the code examples and exercises.

What you will learn
  • Router: Delegate requests to the right specialist agent or path.
  • Planner → Executor: Use a capable model to plan, and a fast model to execute.
  • Self-critique: Add an iterative review loop with a critic to improve outputs.
  • Human-in-the-Loop (HITL): Insert human approval for critical decisions.
  • Wrap-up: Consolidate best practices and next steps.
Prerequisites

You need Python and the Google ADK library installed. ADK is a code-first toolkit for building, evaluating, and deploying AI agents.

1. Create and activate a virtual environment. 2. Install Google ADK: `pip install google-adk` 3. Create a `main.py` file to experiment with the snippets in this course. Get your Gemini API key and environment setup from Google AI Studio. Many ADK examples use Gemini models like gemini-2.0-flash.

In the next stage, you will implement the Router pattern to delegate tasks to the right specialist automatically.
