The MCP Compass enables GitHub Copilot to discover and recommend appropriate Model Context Protocol servers based on your needs. This intelligent discovery tool helps you find the right MCPs for specific tasks through natural language queries, making it easier to navigate the growing ecosystem of MCP services.

Features:
  • Find MCP services using natural language queries
  • Get detailed information about available MCPs
  • Receive recommendations based on your specific needs
  • Stay up-to-date with the latest MCP services
  • Learn how to use and configure different MCPs

Key Use Cases
  • Discovering MCPs for specific programming tasks
  • Learning about alternative tools for your workflow
  • Finding the right combination of MCPs for a project
  • Exploring the capabilities of the MCP ecosystem

Setup Instructions

Setting up MCP Compass is straightforward with NPX:

Configuration
Create or edit a file .vscode/mcp.json to include the following MCP setup: { "servers": { "mcp-compass": { "command": "npx", "args": [ "-y", "@liuyoshio/mcp-compass" ] } } } If mcp.json file is already created, just add "mcp-compass" object to the "servers" array and don't remove existing servers.
